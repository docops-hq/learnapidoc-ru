–î–æ —Ç–æ–≥–æ, –∫–∞–∫ REST —Å—Ç–∞–ª —Å–∞–º—ã–º –ø–æ–ø—É–ª—è—Ä–Ω—ã–º –≤–µ–±-—Å–µ—Ä–≤–∏—Å–æ–º, SOAP (Simple Object Access Protocol) –±—ã–ª –≥–æ—Ä–∞–∑–¥–æ –±–æ–ª–µ–µ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–Ω—ã–º. –ß—Ç–æ–±—ã –ª—É—á—à–µ –ø–æ–Ω—è—Ç—å REST, –ø–æ–ª–µ–∑–Ω–æ –∏–º–µ—Ç—å –Ω–µ–∫–æ—Ç–æ—Ä–æ–µ –ø–æ–Ω–∏–º–∞–Ω–∏–µ SOAP. –ó–∞–æ–¥–Ω–æ –∏ —É–≤–∏–¥–µ—Ç—å, —á—Ç–æ –æ—Ç–ª–∏—á–∞–µ—Ç SOAP –∏ REST.


SOAP - —ç—Ç–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –ø—Ä–æ—Ç–æ–∫–æ–ª, –∫–æ—Ç–æ—Ä—ã–π —Ç—Ä–µ–±—É–µ—Ç XML –≤ –∫–∞—á–µ—Å—Ç–≤–µ —Ñ–æ—Ä–º–∞—Ç–∞ —Å–æ–æ–±—â–µ–Ω–∏–π –¥–ª—è –∑–∞–ø—Ä–æ—Å–æ–≤ –∏ –æ—Ç–≤–µ—Ç–æ–≤. –ö–∞–∫ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π –ø—Ä–æ—Ç–æ–∫–æ–ª, —Ñ–æ—Ä–º–∞—Ç —Å–æ–æ–±—â–µ–Ω–∏—è –æ–±—ã—á–Ω–æ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è —Å –ø–æ–º–æ—â—å—é —Ñ–∞–π–ª–∞ WSDL (—è–∑—ã–∫ –æ–ø–∏—Å–∞–Ω–∏—è –≤–µ–±-—Å–ª—É–∂–±).

–û—Å–Ω–æ–≤–Ω–∞—è –ø—Ä–æ–±–ª–µ–º–∞ —Å SOAP –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ —Ç–æ–º, —á—Ç–æ —Ñ–æ—Ä–º–∞—Ç —Å–æ–æ–±—â–µ–Ω–∏–π XML —Å–ª–∏—à–∫–æ–º –º–Ω–æ–≥–æ—Å–ª–æ–≤–Ω—ã–π –∏ —Ç—è–∂–µ–ª—ã–π. –û—Å–æ–±–µ–Ω–Ω–æ —ç—Ç–æ –ø—Ä–æ–±–ª–µ–º–∞—Ç–∏—á–Ω–æ –≤ –º–æ–±–∏–ª—å–Ω—ã—Ö —Å—Ü–µ–Ω–∞—Ä–∏—è—Ö, –≥–¥–µ —Ä–∞–∑–º–µ—Ä —Ñ–∞–π–ª–∞ –∏ –ø—Ä–æ–ø—É—Å–∫–Ω–∞—è —Å–ø–æ—Å–æ–±–Ω–æ—Å—Ç—å –∏–≥—Ä–∞—é—Ç —Ä–µ—à–∞—é—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ. –ú–Ω–æ–≥–æ—Å–ª–æ–≤–Ω—ã–π —Ñ–æ—Ä–º–∞—Ç —Å–æ–æ–±—â–µ–Ω–∏—è –∑–∞–º–µ–¥–ª—è–µ—Ç –≤—Ä–µ–º—è –æ–±—Ä–∞–±–æ—Ç–∫–∏, —á—Ç–æ –¥–µ–ª–∞–µ—Ç –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏—è SOAP –≤—è–ª—ã–º.

SOAP –ø–æ-–ø—Ä–µ–∂–Ω–µ–º—É –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤ enterprise –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è—Ö (–æ—Å–æ–±–µ–Ω–Ω–æ –≤ —Ñ–∏–Ω–∞–Ω—Å–æ–≤—ã—Ö —É—á—Ä–µ–∂–¥–µ–Ω–∏—è—Ö) —Å–æ —Å–≤—è–∑—å—é server-to-server, –Ω–æ –≤ –ø–æ—Å–ª–µ–¥–Ω–∏–µ –≥–æ–¥—ã SOAP –≤—ã–º–µ—â–∞–µ—Ç—Å—è REST, –æ—Å–æ–±–µ–Ω–Ω–æ –¥–ª—è API –æ—Ç–∫—Ä—ã—Ç—ã—Ö —Å–µ—Ç–µ–π.

## REST - —Å—Ç–∏–ª—å, –∞ –Ω–µ —Å—Ç–∞–Ω–¥–∞—Ä—Ç

–ö–∞–∫ –∏ SOAP, REST (Representational State Transfer) –∏—Å–ø–æ–ª—å–∑—É–µ—Ç HTTP –≤ –∫–∞—á–µ—Å—Ç–≤–µ –ø—Ä–æ—Ç–æ–∫–æ–ª–∞ –ø–µ—Ä–µ–¥–∞—á–∏ –¥–∞–Ω–Ω—ã—Ö –¥–ª—è –∑–∞–ø—Ä–æ—Å–æ–≤ –∏ –æ—Ç–≤–µ—Ç–æ–≤. –û–¥–Ω–∞–∫–æ, –≤ –æ—Ç–ª–∏—á–∏–µ –æ—Ç SOAP, REST —è–≤–ª—è–µ—Ç—Å—è –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–º —Å—Ç–∏–ª–µ–º, –∞ –Ω–µ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–º –ø—Ä–æ—Ç–æ–∫–æ–ª–æ–º. –í–æ—Ç –ø–æ—á–µ–º—É REST API –∏–Ω–æ–≥–¥–∞ –Ω–∞–∑—ã–≤–∞—é—Ç RESTful API - REST - —ç—Ç–æ –æ–±—â–∏–π —Å—Ç–∏–ª—å, –∫–æ—Ç–æ—Ä–æ–º—É —Å–ª–µ–¥—É–µ—Ç API.

API-–∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å RESTful –º–æ–∂–µ—Ç –Ω–µ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–æ–≤–∞—Ç—å –≤—Å–µ–º –æ—Ñ–∏—Ü–∏–∞–ª—å–Ω—ã–º —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∞–º REST, —É–∫–∞–∑–∞–Ω–Ω—ã–º [–¥–æ–∫—Ç–æ—Ä–æ–º –†–æ–µ–º –§–∏–ª–¥–∏–Ω–≥–æ–º](https://en.wikipedia.org/wiki/Roy_Fielding), –∫–æ—Ç–æ—Ä—ã–π –≤–ø–µ—Ä–≤—ã–µ –æ–ø–∏—Å–∞–ª —ç—Ç—É –º–æ–¥–µ–ª—å. –°–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, API —è–≤–ª—è—é—Ç—Å—è ¬´RESTful¬ª –∏–ª–∏ ¬´REST-–ø–æ–¥–æ–±–Ω—ã–º–∏¬ª. (–ù–µ–∫–æ—Ç–æ—Ä—ã–µ —Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–∏ –Ω–∞—Å—Ç–∞–∏–≤–∞—é—Ç –Ω–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ —Ç–µ—Ä–º–∏–Ω–∞ ¬´RESTful¬ª, –∫–æ–≥–¥–∞ API –Ω–µ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –≤—Å–µ–º —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∞–º REST, –Ω–æ –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ –ª—é–¥–µ–π –ø—Ä–æ—Å—Ç–æ –Ω–∞–∑—ã–≤–∞—é—Ç –∏—Ö ¬´REST API¬ª)

–í –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω–æ–º —Å—Ç–∏–ª–µ –Ω–µ—Ç –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–π XML –≤ –∫–∞—á–µ—Å—Ç–≤–µ —Ñ–æ—Ä–º–∞—Ç–∞ —Å–æ–æ–±—â–µ–Ω–∏—è. API REST –º–æ–≥—É—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –ª—é–±–æ–π —Ñ–æ—Ä–º–∞—Ç —Å–æ–æ–±—â–µ–Ω–∏–π, –∫–æ—Ç–æ—Ä—ã–π —Ö–æ—Ç—è—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–∏ API, –≤–∫–ª—é—á–∞—è XML, JSON, Atom, RSS, CSV, HTML –∏ –¥—Ä—É–≥–∏–µ.

–ù–µ—Å–º–æ—Ç—Ä—è –Ω–∞ —Ä–∞–∑–Ω–æ–æ–±—Ä–∞–∑–∏–µ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ —Ñ–æ—Ä–º–∞—Ç–∞ —Å–æ–æ–±—â–µ–Ω–∏–π, –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ API REST –∏—Å–ø–æ–ª—å–∑—É—é—Ç JSON (–Ω–æ—Ç–∞—Ü–∏—é –æ–±—ä–µ–∫—Ç–æ–≤ JavaScript) –≤ –∫–∞—á–µ—Å—Ç–≤–µ —Ñ–æ—Ä–º–∞—Ç–∞ —Å–æ–æ–±—â–µ–Ω–∏–π –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é. –û–Ω–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç JSON, –ø–æ—Ç–æ–º—É —á—Ç–æ –æ–Ω –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –ª–µ–≥–∫–∏–π, –ø—Ä–æ—Å—Ç–æ–π –∏ –±–æ–ª–µ–µ –≥–∏–±–∫–∏–π —Ñ–æ—Ä–º–∞—Ç —Å–æ–æ–±—â–µ–Ω–∏–π, –∫–æ—Ç–æ—Ä—ã–π —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç —Å–∫–æ—Ä–æ—Å—Ç—å —Å–≤—è–∑–∏. –û–±–ª–µ–≥—á–µ–Ω–Ω–∞—è –ø—Ä–∏—Ä–æ–¥–∞ JSON —Ç–∞–∫–∂–µ –ø–æ–∑–≤–æ–ª—è–µ—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å —Å—Ü–µ–Ω–∞—Ä–∏–∏ –º–æ–±–∏–ª—å–Ω–æ–π —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–∏ –∏ –ª–µ–≥–æ–∫ –¥–ª—è –ø–∞—Ä—Å–∏–Ω–≥–∞ –≤ –ò–Ω—Ç–µ—Ä–Ω–µ—Ç–µ —Å –ø–æ–º–æ—â—å—é JavaScript.

–ù–∞–ø—Ä–æ—Ç–∏–≤, –≤ XML, XSLT –±–æ–ª—å—à–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è –∏–ª–∏, —Å–∫–æ—Ä–µ–µ, ¬´–ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è¬ª (¬´T¬ª –≤ XSLT) —Å–æ–¥–µ—Ä–∂–∏–º–æ–≥–æ, —Ö—Ä–∞–Ω—è—â–µ–≥–æ—Å—è –Ω–∞ —è–∑—ã–∫–µ XML. XSLT –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç —É–¥–æ–±–æ—á–∏—Ç–∞–µ–º–æ—Å—Ç—å (–∞ –Ω–µ –æ–±—Ä–∞–±–æ—Ç–∫—É –¥–∞–Ω–Ω—ã—Ö, —Ö—Ä–∞–Ω—è—â–∏—Ö—Å—è –≤ —Ñ–æ—Ä–º–∞—Ç–µ XML).

### REST —Ñ–æ–∫—É—Å–∏—Ä—É–µ—Ç—Å—è –Ω–∞ —Ä–µ—Å—É—Ä—Å–∞—Ö, –¥–æ—Å—Ç—É–ø –∫ –∫–æ—Ç–æ—Ä—ã–º –æ—Å—É—â–µ—Å—Ç–≤–ª—è–µ—Ç—Å—è —á–µ—Ä–µ–∑ URL

–î—Ä—É–≥–æ–π –æ—Ç–ª–∏—á–∏—Ç–µ–ª—å–Ω–æ–π —á–µ—Ä—Ç–æ–π REST —è–≤–ª—è–µ—Ç—Å—è —Ç–æ, —á—Ç–æ API-–∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å—ã REST —Ñ–æ–∫—É—Å–∏—Ä—É—é—Ç—Å—è –Ω–∞ —Ä–µ—Å—É—Ä—Å–∞—Ö (—Ç–æ –µ—Å—Ç—å –≤–µ—â–∞—Ö, –∞ –Ω–µ –¥–µ–π—Å—Ç–≤–∏—è—Ö) –∏ —Å–ø–æ—Å–æ–±–∞—Ö –¥–æ—Å—Ç—É–ø–∞ –∫ —Ä–µ—Å—É—Ä—Å–∞–º. –†–µ—Å—É—Ä—Å—ã, –∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ, —è–≤–ª—è—é—Ç—Å—è —Ä–∞–∑–Ω—ã–º–∏ —Ç–∏–ø–∞–º–∏ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏. –í—ã –ø–æ–ª—É—á–∞–µ—Ç–µ –¥–æ—Å—Ç—É–ø –∫ —Ä–µ—Å—É—Ä—Å–∞–º —á–µ—Ä–µ–∑ URL (Uniform Resource Locators), —Ç–∞–∫ –∂–µ –∫–∞–∫ –ø–µ—Ä–µ—Ö–æ–¥ –∫ URL-–∞–¥—Ä–µ—Å—É –≤ –≤–∞—à–µ–º –±—Ä–∞—É–∑–µ—Ä–µ –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–æ–¥–∫–ª—é—á–∏—Ç—å—Å—è –∫ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–æ–Ω–Ω–æ–º—É —Ä–µ—Å—É—Ä—Å—É.
