При использовании Swagger UI, вывод Swagger UI не обязательно должен быть автономным сайтом. Можно встроить Swagger в существующую веб-страницу. Ниже приведен встроенный экземпляр Swagger UI показывающий файл OpenAPI file for the OpenWeatherMapAPI.

Дисплей пользовательского интерфейса Swagger спроектирован так, чтобы быть отзывчивым, разделы свертывания / развертывания в представлениях модели по-прежнему могут переполняться в представлениях, поэтому при встраивании можно столкнуться с некоторыми проблемами.
